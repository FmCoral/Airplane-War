## 玩法介绍

### 环境布置

**1**、安装Python环境
- [下载Python](https://www.python.org/downloads/)

**2**、安装Pygame库

```bash
pip install pygame
```

**3**、安装git

- [下载git](https://git-scm.com/downloads)

**4**、克隆项目仓库

```bash
git clone https://github.com/FmCoral/Airplane-War.git 
```

### 开始游戏

**1**、进入项目目录

```bash
cd airplane-war
```

**2**、运行游戏
```bash
python main.py
```
